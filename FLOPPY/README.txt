Floppy Disk experiments.

BOOTME.PAS Based on COMASM, creates an experimental boot sector for floppy discs.

NULLDISK.PAS Utility to wipe boot sector of a floppy.

READBOOT.PAS Utility to read and save a floppy boot sector.

WRITBOOT.PAS Writes a specified file to a floppy boot sector.
